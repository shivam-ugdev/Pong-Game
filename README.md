# Pong (Unity Practice Project)
A recreation of the classic Pong game developed using Unity and C#.
This project focuses on fundamental game development concepts such as player input, ball movement, collision handling, score tracking, and basic game logic. It was created as a practice project to strengthen core 2D game development skills.
## Features
- Classic Pong gameplay
- Player and AI paddle controls
- Ball movement and collision detection
- Score tracking system
- Simple game logic
